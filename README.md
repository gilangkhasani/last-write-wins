## Tech Stack:
* Language: Java 11
* Framework: Spring Boot
* Build Tool: Maven

## Install & Run:
* Install Java 11.
* Install Maven 3
* git clone https://github.com/randomvlad/TicTacToe.git
* Run with mvn spring-boot:run
* To terminate mvn spring-boot:run, kill process with CTRL + C.
* Run the test using mvn clean test
